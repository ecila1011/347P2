Endpoints

Posting:

/first_list and /second_list

In the example curl function below, you would send in data as "item" and then the word you'd like to add to the list

// curl --header 'Content-Type: application/json' --data '{"item" : "bird"}' http://localhost:5001/first_list

Get:

/first_list prints all of the items in the first list

/second_list prints all of the items in the second list

/first_list/:id prints the item from the first list with the specified id

/second_list/:id prints the item from the second list with the specified id

Patch:

/first_list/:id updates the spelling of a item in the first list with the specified id

/second_list/:id updates the spelling of a item in the second list with the specified id

Delete:

/first_list/:id deletes the item with the specified id

/second_list/:id deletes the item with the specified id
